## 🎨 Cómo Crear el Carousel en Canva

### Paso 1: Setup
1. Ir a Canva.com
2. Crear diseño personalizado: 1080x1080px
3. Crear 10 páginas (slides)

### Paso 2: Diseño
Use la guía en `security-fixes-carousel.md`:
- **Color palette:** Dark blue (#1a1f36) + Electric blue (#00d4ff)
- **Fuente headers:** Montserrat Bold, 48px
- **Fuente body:** Inter Regular, 32px
- **Fuente code:** Fira Code, 24px
- **Padding:** 80px en todos los lados

### Paso 3: Contenido
Copia el texto de cada slide del archivo markdown directamente a Canva.

### Paso 4: Export
- Formato: PNG
- Resolución: Alta (300 DPI)
- Comprimir ligeramente para LinkedIn (<5MB total)

## ✍️ Cómo Publicar en LinkedIn

### Para Carousel:
1. Crear nuevo post
2. Click en "📸 Photo"
3. Seleccionar las 10 imágenes en orden
4. Copiar caption de `security-fixes-carousel.md`
5. Agregar hashtags
6. **Tip:** Poner "Slide 1/10 →" en el caption para que la gente sepa que es carousel

### Para Post Corto:
1. Crear nuevo post
2. Copiar texto de `security-fixes-short.md`
3. Formatear code blocks con:
   ```
   Código aquí (usa fuente monospace)
   ```
4. Agregar emojis para visual interest
5. Hashtags al final

### Para Artículo:
1. Click en "Write article"
2. Copiar contenido de `security-fixes-agent-tokens.md`
3. Agregar imágenes de los code snippets
4. Publicar con preview atractivo
